github 博客建站经历

第一篇文章, 时间不早了, 浅浅写写

  1. 拥有一个 github 账号

  2. 新建项目 名为 “{username}.github.io”, 额外建立分支为 gh-pages

  3. 本地上安装 hexo (或者其他博客网站模版)

  4. hexo init 后提交到 github 上 sitting -> pages -> source -> actions 自己建立工作流程

  5. pages.yml内容如下

  6. 设置 sitting -> pages -> source -> Deploy from a branch, Branch 选择gh-pages, 使用根目录

  7. 从此之后每次 push 到 main 分支都会执行任务, github action会自动将构建出来的 public 文件夹 push 到 gh-pages 分支中

  8. 保存
